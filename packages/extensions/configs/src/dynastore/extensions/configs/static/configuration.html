No “Asset” scope. Configuration tiers are platform → catalog → collection only. Asset behaviour is configured through plugin classes such as AssetRoutingConfig at one of those three tiers — there is no separate asset radio. Each field’s from badge shows which tier supplied its effective value; the Diff and Wire tabs preview and explain the exact PATCH Save will send.
