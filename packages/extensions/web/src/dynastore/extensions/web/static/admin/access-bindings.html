Bindings are the third axis of authorization: who, on what resource, with what role or policy. Platform and catalog roles live in the Governance hub; this page is for the per-collection bindings — the finer scope where most real-world access is shaped.

Attribute stamping (ABAC)

GET / PUT /configs/catalogs/{cat}/collections/{col}/plugins/attribute_stamping_policy

Declare which Feature properties are lifted onto the document's _attrs envelope at write time, enabling per-document ABAC enforcement via attribute_predicates on grants. Only $.properties.<field> paths are supported in this slice.
