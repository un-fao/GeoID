# Agro-Informatics Platform: Catalog Services Manual

## Introduction
The Agro-Informatics Platform (AIP) - Catalog Services is a state-of-the-art, cloud-native geospatial infrastructure. It provides a modular framework for managing, processing, and serving large-scale geospatial datasets using OGC standards.

## Key Features
- **True Multi-Tenancy**: Isolated environments for different catalogs.
- **Extreme Scalability**: Database-driven partitioning and horizontal scaling.
- **OGC Compliant**: Native support for OGC API - Features, STAC, Tiles, and Processes.
- **Modular Architecture**: "Three Pillars" design (Modules, Extensions, Tasks) for maximum flexibility.

## Getting Started
To begin using the platform, explore the following areas:
- **Platform & Customers**: Overview of enterprise capabilities.
- **Architecture**: Deep dive into the "Three Pillars" and storage abstraction.
- **APIs**: Access raw data via standardized OGC interfaces.

## For Developers
The platform exposes the following endpoints:
- **OpenAPI / Swagger**: Interactive API documentation for exploring and testing endpoints.
- **STAC Catalog**: Discover and browse geospatial assets.
- **OGC Features**: Query and retrieve vector data.
